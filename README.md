🌏 *A vibrant student community dedicated to **celebrating Bangladeshi culture**,*  
*fostering friendships, and creating unforgettable memories for students in Sydney.*

💻 *Our GitHub is home to **open-source projects, digital archives, and cultural resources***  
*— built by students, for students.*
